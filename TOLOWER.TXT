ToLower version 1.1
-------------------
This is a small program that was hacked together to convert all files in the current directory to lower case. This is usefull when uploading something to a unix (web) server.


Usage:
-----------------
Run "tolower.exe" and it will turn all the files on that directory to lower case.  


Compile tools:
-----------------
* IBM C and C++ Compilers, Version 3.6.5 - FP 2. (ICC.EXE)
* NMake tool version 4.00.001.


History revision:
-----------------
- v1.0 First version. (1997-09-21)
- v1.1 Recompiled on 2021. (2021-02-02)


Story:
-----------------
Tels had downloaded once a program that did this, but it destroyed some files badly and damaged most off the filenames. So he thought about writing some small tool for himself.
